## Installation
- Install dependencies with:
```bash
poetry install
```

## Commands
### Run program 
- Run with:
```bash
poetry run invoke start
```

### Testing 
- Run tests with:
```bash 
poetry run invoke test
```

### Test coverage
- Run coverage tests:
```bash 
poetry run invoke coverage
```
- Generate test coverage raport with:
```bash
poetry run invoke coverage-report
```

### Pylint
- Run pylint with:
```bash
poetry run invoke lint
```
